# ChenStack 功能对照

参考页面：

- https://www.chenstack.cn/2024/01/21/palworld/
- 页面标题为“幻兽帕鲁(Palworld)修改器”，发布时间为 `2024-01-21`，页面显示更新于 `2025-06-14`

这份对照表的目的不是逐项照搬，而是把外部现有功能拆成三类：

1. 我们已经有基础能力，适合继续做深
2. 适合继续做，但限定在房主 / 单机 / 客户端可见范围
3. 不走越权和主机权限提升路线

## 已有基础能力，可继续做深

- 物品、帕鲁、科技相关的房主命令生成
- 传送、时间、经验等房主命令模板
- 运行时扫描、附近玩家扫描、刷怪点扫描
- 会话日志和书签化扫描工作流

这些已经覆盖了“命令组织层”和“客户端信息层”，下一步更适合往批量预设、路线库、地图书签、收藏点追踪去扩。

## 适合继续做的安全方向

### 非房主也有价值的客户端增强

- 地图坐标书签与路线库
- 会话日志筛选、导出、回放
- 资源点 / 宝箱 / 刷怪点 / NPC 的本地发现面板
- 收藏品追踪：手记、雕像、补给点、Boss 点位
- 扫描预设收藏夹和一键复扫

### 房主 / 单机增强

- 批量物品宏
- 科技包和建筑包
- Spawn 组合包
- 旅行预设库
- 更完整的存档/角色/帕鲁参数编辑器

## 不按这个方向做的内容

下面这些如果要在“非房主联机”里成立，通常就会落到主机权限绕过、服务器权威绕过或对联机规则做强制修改：

- 非房主直接生成物品、帕鲁、科技
- 非房主直接改玩家或帕鲁服务端数据
- 非房主直接解锁传送点、建筑限制、终端限制
- 任何隐藏管理员能力、权限提升、反作弊绕过

这部分不作为本项目路线。

## 基于这份对照的下一阶段优先级

- P1：地图书签、路线库、收藏点追踪
- P1：会话日志查看器和扫描回放
- P2：房主批量命令预设和导入导出
- P2：更细的 Palworld 专项扫描面板
